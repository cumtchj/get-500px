##安装<a href="http://nodejs.cn/download/">node.js</a>

###测试node.js安装是否成功
```
# windows操作系统
// win+R打开运行窗口，输入cmd打开命令行
node -v
npm -v

# mac操作系统
// 打开控制台
node -v
npm -v

```


##使用方式

###1.安装依赖

```
npm install
```

###2.启动项目

```
npm run start
```

###3.浏览器内访问链接
localhost:7010/500get/index.html
